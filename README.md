# Self-balancing robot

BLE-controlled self-balancing robot

- [Android BLE Joystick](/android/serial-ble-rc)
- [Arduino Code](/nano/twbr-nano)

## BOM

- Keywish BLE Nano x1
- A4988 Stepper Driver x2
- Stepper motor Nema17 x2
- MPU6050 x1
